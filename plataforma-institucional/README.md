# Plataforma institucional Voluntários da Alegria

## Como rodar

A plataforma é um site feito de maneira estática, ultilizando apenas HTML e CSS. É possível rodar a plataforma tanto arrastando o arquivo index.html em seu navegador de preferência, ou, caso esteja ultilizando o VSCode, recomenda-se a instalação da instenção [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer).

### Como abrir direto no navegador

- Abra a pasta "plataforma-institucional" no seu explorador de arquivos
- Arraste o arquivo index.html em seu navegador de preferência

### Como abrir com o Live Server

- Clicar com o botão direito do mouse no arquivo index.html
- Clicar na opção <strong>"Open with Live Server"</strong>

### Observações

- Para desenvolvimento, é recomendado usar a opção com o Live Server ou equivalentes, porque ao atualizar o código, a página é atualizada automaticamente.
